📌 什么是 DeepSeek OCR？
这是一个强大的 AI 图像文字识别工具，可以：
✓ 识别图片中的文字并转为可编辑文本
✓ 将文档图片转换为 Markdown 格式
✓ 支持复杂排版和多语言识别
✓ 完全离线运行（首次需要联网下载模型）

==================================================
🚀 快速开始（3步搞定）
==================================================

第一步：启动程序
  → 双击 DeepSeekOCR.exe

第二步：加载模型（仅首次需要）
  → 点击界面上的"加载模型"按钮
  → 等待约 5-15 分钟（会自动下载 AI 模型）
  → 看到"状态：已加载"（绿色）即可

第三步：识别图片
  → 点击"选择图像"选择你的图片
  → 点击"开始识别"
  → 等待识别完成，结果会显示在下方

==================================================
⚙️ 系统要求
==================================================

最低配置：
  • Windows 10 (64位)
  • 8GB 内存
  • 10GB 可用磁盘空间
  • 首次运行需要网络连接

推荐配置：
  • Windows 10/11 (64位)
  • 16GB 或更多内存
  • NVIDIA 显卡（GPU 会快很多）
  • 20GB 可用磁盘空间

==================================================
💡 使用技巧
==================================================

1. 预设配置选择：
   • Gundam（推荐）：平衡速度和质量
   • Tiny/Small：速度快，质量较低
   • Base/Large：质量高，速度较慢

2. 提示词说明：
   • 默认提示词可以识别并转为 Markdown
   • 简单识别用："<image>\nFree OCR. "

3. 输出目录：
   • 默认保存在程序的 output 文件夹
   • 可以自定义到任何位置

❓ 常见问题
==================================================

Q1: 首次运行需要多久？
A1: 首次需要下载模型（5-10 GB），根据网速需要 
    10-30 分钟。之后运行不需要再下载。

Q2: 能否离线使用？
A2: 可以！模型下载后，完全可以离线使用。

Q3: 识别速度慢怎么办？
A3: • 如果有 NVIDIA 显卡，确保安装了正确的驱动
    • 使用较小的预设配置（Tiny/Small）
    • 关闭其他占用内存的程序

Q4: 提示"CUDA 不可用"
A4: 表示没有检测到 NVIDIA GPU，程序会使用 CPU
    运行。虽然较慢但功能完全正常。

Q5: 模型下载失败
A5: • 检查网络连接是否稳定
    • 确认防火墙没有阻止程序
    • 可以换个时间段再试

Q6: 识别结果不准确
A6: • 尝试使用 Large 预设（质量更高）
    • 确保图片清晰、光线充足
    • 避免图片过于模糊或倾斜

Q7: 杀毒软件报警
A7: 这是误报，可以添加到白名单。程序完全安全。

⚠️ 重要提示
==================================================

1. 首次运行请确保：
   • 有稳定的网络连接
   • 至少 15 GB 可用磁盘空间
   • 不要在下载模型时关闭程序

2. 模型存储位置：
   C:\Users\你的用户名\.cache\huggingface\
   （卸载程序不会删除模型）

3. 性能建议：
   • 推荐使用 SSD 硬盘
   • 关闭不必要的后台程序
   • 有 GPU 的电脑体验最佳
